# AI-Powered RoboTaxi Simulation

Multi-agent taxi dispatch simulation implementing intelligent pathfinding and fare allocation algorithms for COMP6032 coursework.

## About

This project implements an AI-driven taxi dispatch system featuring autonomous taxi agents, dynamic fare generation across a city grid network, centralized dispatcher for fare allocation, and real-time visualization.

## Requirements

- Python 3.8 or higher
- pygame 2.5.0+
- numpy 1.24.0+

## Installation

Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the simulation:
```bash
python RoboUber.py
```
